This project is the implementation of the duckflux runner, which executes duckflux workflows. The runner is a CLI tool that takes a workflow YAML file as input and executes it according to the defined steps and participants.

You can understand the project by simply reading the README, but if you need to dive into the initial implementation plan and rationale, check out the docs/IMPLEMENTATION_PLAN.md file.

Treat the file docs/HISTORY.md as your memory for the project. It contains a chronological record of all significant decisions, changes, and milestones in the development of duckflux. Follow the past decisions to decide on the future direction of the project.
